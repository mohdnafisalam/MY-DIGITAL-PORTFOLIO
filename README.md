# Personal Digital Portfolio

Welcome to my personal digital portfolio repository! This website is built entirely from scratch using semantic **HTML5** and custom **CSS3** to showcase my academic background, technical skill set, and software engineering projects.

---

## 🚀 Live Demo
👉 [View My Live Portfolio](https://mohdnafisalam.github.io/MY-DIGITAL-PORTFOLIO/)

---

## ✨ Features

* **Multi-Page Architecture:** Clean structural separation across dedicated pages for optimal organization.
* **Modular CSS Design:** Features a dedicated stylesheet for nearly every sub-page (e.g., `skills.css`, `projects.css`, `contact.css`) to maintain clean, isolated, and highly readable styling logic.
* **Semantic Code Base:** Built strictly with semantic HTML5 structural elements (`<header>`, `<nav>`, `<main>`, `<section>`) for superior accessibility and SEO practices.
* **Interactive Navigation:** A responsive navigation bar utilizing active-state configurations to streamline user experience.

## 📂 Project Structure & Pages

As seen in the workspace directory layout, the project is cleanly separated into core assets and content subfolders with a root-level landing entry point:

```text
PROJECT 3/
├── CSS/
│   ├── about.css       # Layout styles for the About page
│   ├── contact.css     # Form and contact link configurations
│   ├── demo.css        # Styles specific to the live demo page

│   ├── skills.css      # Progress bars, grids, or badge styling for technical skills

├── HTML/
│   ├── about.html      # Detailed background history and biography

│   └── skills.html     # Comprehensive technical skill matrix
